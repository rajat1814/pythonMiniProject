# 📊 Student Performance Analysis
### Mini Project | Programming in Python | Group: ABC-VI-B

A data analysis and visualization project using **Pandas** and **Matplotlib** to analyze student performance across multiple subjects.

---

## 🔍 Features
- Loads student marks data using Pandas DataFrame
- Calculates **Total**, **Average**, and **Grade** for each student
- Generates a **4-panel dashboard** with:
  - Bar Chart — Average score per student (color-coded by grade)
  - Line Chart — Subject-wise score trends
  - Pie Chart — Grade distribution across the class
  - Horizontal Bar Chart — Subject-wise class averages
- Exports results to **CSV**

---

## 🛠️ Technologies Used
| Tool | Purpose |
|------|---------|
| Python 3 | Core programming language |
| Pandas | Data loading, manipulation, analysis |
| Matplotlib | Data visualization / charts |

## ▶️ How to Run

### 1. Clone the repository
```bash
git clone https://github.com/your-username/student-performance-analysis.git
cd student-performance-analysis
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the analysis
```bash
python analysis.py
```

Output files will be saved in the `output/` folder:
- `output/dashboard.png` — Visualization dashboard
- `output/results.csv` — Full results table
